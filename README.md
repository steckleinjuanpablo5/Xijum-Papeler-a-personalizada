# Xijúm Store

Arquitectura base del proyecto:

- `frontend/`: sitio estático para GitHub Pages
- `backend/`: API Node/Express para Render
- pagos: Stripe Checkout desde backend
- secretos: solo en Render Environment Variables

## Fases
1. Arquitectura y setup
2. Frontend base estable
3. Catálogo y filtros
4. Carrito
5. Resumen de compra
6. Backend + Stripe
7. Deploy y validación
8. Diseńo visual final
